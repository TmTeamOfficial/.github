# TmTeamOfficial

Official GitHub account of TmTeam company (tm-team.ru)

## Our projects

- FindTM—the program for the Windows operating system written in Python that was parsing the search for you and showed only sites (links) that would open according to your internet connection 

- DownloadTM—the bot in ICQ written in Python that helped users download videos from YouTube

- Vpn2App - An application for Android written in Flutter/Dart that was using Google Drive as a proxy for communication between the server and the user.

- Vpn2App_bot - A bot in Telegram written in Python that was parsing the channels with free proxies and key files and sending them to Google Drive

- FindTM_ext - Extension for Chrome Browser written in JavaScript that was helping users to save time by automatically looking inside the links in Google Search and marking them by accessibility
